Возможности кейлогера на момент 03.12.2019(версия SystemDrivers_15):
	Добавление в автозагрузку всем возможным уникальным пользователям(т.е тем пользователям, которые были созданы и к которым есть доступ) и активация флага "скрытый"
	Отпрвка на почту файлов по электронной почте данных кейлогера текущего пользователя(с которого запущен кейлогер)
	Создание сеансов для каждого пользователя(создание файла данных и файла настроек отправки по электронной почте в папке с названием именем пользователя)
	Шифрование данных(сразу записываются данные в шифрованном виде) c использованием бинарного поиска(выборка данных для шифрования намнго быстрее(вместо 156 итераций, всего 8))
	Создание файлов данных на каждый новый день с названием = имя компьютера + число + месяц + .txt
	Перезапись файла настроек отправки по электронной почте каждый раз, когда файл с данными кейлогеа отправляется на указанную электрнную почту
	Текущая дата отправки данных кейлогера по электронной почте выставляется как Falsе, после отпрвки данных или после создания файла
	Программа содает изначально два процесса, один из них висит и ничего не делает, реализована функция его завершения
	Программа создает папку на каждый месяц(куда записываются файлы с данными за текущий месяц и записывается/перезаписывается файл с настройками)
	Название процесса теперь System Drivers.exe, а не SystemDrivers.exe
